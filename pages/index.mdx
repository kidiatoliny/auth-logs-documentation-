## Laravel Auth Logs

This package provides an efficient and flexible way to implement a **authentication logs** in your Laravel application.


## Why Use Laravel Auth Logs?
Laravel Auth Logs is designed to help you track user authentication events in your application. Here are some key benefits:
- **Easy Integration**: Seamlessly integrates with your existing Laravel application.
- **Comprehensive Logging**: Automatically logs user login, logout, and failed login attempts.
- **Customizable**: Easily customize the logging behavior to suit your application's needs.
- **Performance**: Built with performance in mind, ensuring minimal impact on your application's speed.
